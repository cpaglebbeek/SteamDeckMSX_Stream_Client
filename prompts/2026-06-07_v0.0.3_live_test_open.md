## Wat moet v0.0.3 doen

### 1. Moonlight install + first-launch op Deck

```bash
flatpak install flathub com.moonlight_stream.Moonlight
flatpak run com.moonlight_stream.Moonlight  # one-time UI
```

### 2. pair.sh dry-run vóór echte pair

```bash
./scripts/pair.sh --dry-run horsecloud55.ddns.net
```

Verwacht: 6 stappen tonen wat gebeurt, geen actie.

### 3. pair.sh echte pair

```bash
./scripts/pair.sh horsecloud55.ddns.net
# Vervolgens: browser openen op https://horsecloud55.ddns.net:47990
# 4-cijferige PIN invoeren
# pair.sh wacht en verifieert via app-list
```

Verwacht: "✓ Pair OK — MSX-apps zichtbaar" eind-output.

### 4. measure-latency.sh

```bash
./scripts/measure-latency.sh horsecloud55.ddns.net
./scripts/measure-latency.sh horsecloud55.ddns.net --count 30
```

Verwacht: VERDICT GOED (≤30ms) op residential glasvezel. Bij VERDICT SLECHT: route-debug (traceroute, MTR).

### 5. msx-gamepad.vdf importeren in Steam Input

- Steam app op Deck (Gaming Mode)
- Settings → Controller → Layouts → Browse Configurations
- Import `presets/msx-gamepad.vdf` (via USB-key of `cp ~/scp-target/msx-gamepad.vdf ~/.steam/...`)
- Bind aan "MSX-stream"-app (zichtbaar in Steam-game-lijst na pair)

### 6. scripts/launch-msx.sh — Gaming-Mode launcher

Nieuw script: roep `flatpak run com.moonlight_stream.Moonlight stream <host>:47984 <app-name>` aan met juiste argumenten. Optionele `--app=C-BIOS|BubbleBobble|MetalGear|Nemesis2` flag.

### 7. .desktop file voor "Add Non-Steam Game"

Nieuw: `presets/SteamDeckMSX-Stream.desktop` met `Exec=launch-msx.sh`. Stelt gebruiker in staat `Add a Non-Steam Game` te kiezen in Steam zonder handmatige browse.

### 8. Visuele verificatie preset op Deck

- Open Bubble Bobble via Moonlight stream
- D-pad → joystick beweegt
- A → fire-1, B → fire-2
- L1/R1 → openMSX OSD pijl-cycle (F2/F3) — verifieer dat keys aankomen
- L2/R2 → fast-forward toggle (visuele snelheidstoename)
- Select → openMSX OSD opent

### 9. BUGLIST.md update

Iedere afwijking vastleggen met kleurcode (geel = script-bug, rood = upstream-Moonlight-incompatibiliteit).
