| **`pair.sh` idempotent + dry-run + check-only** | CLAUDE.md zegt dry-run-default in v0.0.3+ — ik trek dit naar v0.0.2 op flag-niveau zodat gebruiker `--dry-run` kan kiezen |

| **Cross-machine via openMSX keymap.txt i.p.v. per-game vdf** | Schaalt beter: 1 vdf-preset voor alle MSX-titels; preset gaat uit van openMSX 21.0 default keys |
